10、设置静态ip地址
    1）VMware->虚拟机->设置->网络适配器->桥接模式->确定
    2）ifconfig查看网卡的名称
    3）cd /etc/netplan
          vi 01-network-manager-all.yaml 
          在network后面添加：
ethernets:
    ens33:                              #配置的网卡的名称
      addresses: [192.168.6.118/24]    #配置的静态ip地址和掩码（lihf：掩码24的意思：前面3字节255,3个字节也就是24bit）
      dhcp4: no                         #关闭DHCP，如果需要打开DHCP则写yes
      optional: true
      gateway4: 192.168.6.1            #网关地址
      nameservers:
         addresses: [192.168.6.1,114.114.114.114]    #DNS服务器地址，多个DNS服务器地址需要用英文逗hao
     上面这段是有格式要求的，如下：
         1、ip地址和DNS服务器地址需要用[]括起来，但是网关地址不需要
         2、注意每个冒号后边都要先加一个空格
         3、注意每一层前边的缩进，至少比上一层多两个空格
     4）netplan apply就ok了

11、安装谷歌拼音输入法
    1）apt-get install fcitx-googlepinyin
    2）配置language support。
         1、win按键，搜索栏输入关键字“language support”，第一次打开会显示语言支持没有完全安装，点击安装并输入密码开始安装
         2、安装完之后，窗口最后一行“keyboard input method system:”，点击下拉菜单，选择刚才安装的“fcitx”
         3、重启电脑。
    3）输入法配置，重启之后，右上角的键盘按钮->configure->Input Method->左下方的“+”号->取消“Only Show Current Language”勾选，搜索框输入pinyin，选择Google Pinyin，点击OK
    4)关闭设置，可以点击右上角键盘切换输入法，快捷键是Ctrl+space

12，安装ssh远程加密连接服务
      1）Ubuntu默认未安装ssh远程加密连接服务，使用如下命令安装即可：apt-get install openssh-server openssh-client，然后重启
      2）如果secureCRT不知道用户名和密码，sudo passwd 用户名 进行修改
      3）重新启动ssh：
           /etc/init.d/ssh stop
           /etc/init.d/ssh restart
      SecureCRT设置（可能需要下载最新的secureCRT版本）：选项->会话选项->连接->SSH2->秘钥交换->我全部勾选（省事）

13、设置共享目录
      1）安装samba，sudo apt-get install samba
      2）/etc/samba/smb.conf 文件最后增加共享目录：
[share]
    comment = share for users
    path = /mnt/share
    browseable = yes
    writable = yes
    public = yes
    directory mask = 0777
    create mask = 0777
    available = yes
      3）service smbd reload  （重新加载参数）
            service smbd stop
            service smbd restart

1、交叉编译json-c
        1）web_server下创建toolChain_json.cmake文件，内容如下：
set(CMAKE_SYSTEM_NAME Linux)
 
set(CMAKE_C_COMPILER arm-linux-gnueabihf-gcc)
set(CMAKE_CXX_COMPILER arm-linux-gnueabihf-g++)
 
set(CMAKE_FIND_ROOT_PATH "/mnt/share/src/web_server/json-c")
set(CMAKE_FIND_ROOT_PATH_MODE_LIBRARY ONLY)
set(CMAKE_FIND_ROOT_PATH_MODE_INCLUDE ONLY)

其中：
TOOLCHAIN_DIR 为交叉编译工具链所在的目录
CMAKE_C_COMPILER 是交叉编译器gcc的名称
CMAKE_CXX_COMPILER 是交叉编译器g++的名称
CMAKE_FIND_ROOT_PATH 是json-c源码存放的路径
以上这些需要根据实际情况填写。

        2）编译
        cd json-c
        cmake -DCMAKE_INSTALL_PREFIX=/mnt/share/build -DCMAKE_TOOLCHAIN_FILE=../toolChain_json.cmake .
        make
        make install
其中-DCMAKE_INSTALL_PREFIX是用来指定编译后库文件的安装路径。

2、编译libubox
        1）web_server下创建toolChain_libubox.cmake文件，内容如下：
set(CMAKE_SYSTEM_NAME Linux)

set(CMAKE_C_COMPILER arm-linux-gnueabihf-gcc)
set(CMAKE_CXX_COMPILER arm-linux-gnueabihf-g++)
 
set(INSTALL_PATH "/mnt/share/build")
include_directories(${INSTALL_PATH}/include)
include_directories(${INSTALL_PATH}/include/json-c)
 
set(CMAKE_FIND_ROOT_PATH "/mnt/share/src/web_server/libubox")
set(CMAKE_FIND_ROOT_PATH_MODE_LIBRARY ONLY)
set(CMAKE_FIND_ROOT_PATH_MODE_INCLUDE ONLY)
 
set(json "/mnt/share/build/lib/libjson-c.so")

其中：
INSTALL_PATH 是上一步骤中编译产生的库文件的安装路径
CMAKE_FIND_ROOT_PATH 是libubox源码的存放路径
json用来指定libjson-c动态库的路径
以上这些根据实际情况填写。

        2）编译
        cd libubox        
        cmake -DBUILD_LUA=OFF -DCMAKE_INSTALL_PREFIX=mnt/share/build -DCMAKE_TOOLCHAIN_FILE=../toolChain_libubox.cmake .
        make
        make install
